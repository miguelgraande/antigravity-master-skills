Contiene herramientas globales de ideacion, produccion, conversion de documentos y utilidades de desarrollo especializadas.

## Como instalar en un proyecto nuevo

Copiar la carpeta `skills/` dentro de `.agent/` de tu proyecto:

```bash
# Clonar y copiar
git clone https://github.com/miguelgraande/antigravity-master-skills
cp -r antigravity-master-skills/skills/* .agent/skills/
```

O descargar solo los skills que necesites directamente desde GitHub.
